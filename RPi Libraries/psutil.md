# Installation of 'psutil'

### Install 'psutil'
```
pip install psutil
```

If you get an error try:
```
sudo apt install python3-psutil
```

If you're using Python 3, you may need to use pip3:
```
pip3 install psutil
```

If you get an error try:
```
sudo apt update
```
```
sudo apt install python3-psutil
```
